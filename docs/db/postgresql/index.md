# PostgreSQLってなに？
## 発音（読み方）
PostgreSQL（ポストグレス キューエル）  
日本では、`ポスグレ` と呼ばれることが多い？

## 種別
関係データベース管理システム（RDBMS）

ほかには、`Oracle` や `MySQL` などがあります。

## シェア
2022年9月現在、`Oracle`、`MySQL`、`SQL Server` の次、四位になってます。
