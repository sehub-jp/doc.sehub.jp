Amazon Linux 2022 初期設定 #

タイムゾーン設定 #

タイムゾーンはデフォルトで UTC なので、東京タイムゾーンに変更します。

現在のタイムゾーン設定を確認 #

bash
timedatectl
1

使用可能のタイムゾーン一覧表示 #

bash
timedatectl list-timezones
1

タイムゾーン設定 #

bash
sudo timedatectl set-timezone Asia/Tokyo
1

「インバウンドルールの編集」を開く #

HTTP と HTTPS のポートを開放 #

ルールを保存 #

「ルールを保存」ボタン押下で http と https でアクセスできるようになります。
